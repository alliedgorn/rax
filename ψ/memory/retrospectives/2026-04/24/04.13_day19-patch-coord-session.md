## Learnings

- **Untracked directories rot silently.** `git status` shows `?? ψ/memory/` every session but it gets habituated. 38 days of brain files were one disk failure from gone. Check for untracked dirs on every pre-rest, not just untracked files.
- **Convergence gates need routing care.** Decree #3 routes all Gorn-action-items through Sable. System-sudo reads as Gorn-weight even when the class is routine-low. Sable initially called TG heads-up; Leonard flagged Prowl-weight via the thread; Sable upgraded cleanly. The framing lane matters, not just the fact of getting Gorn's nod.
- **Autorun scripts need structural verification, not just "success" output.** `patch-status.sh` said "Patch status posted" but the post never landed. Bash string interpolation into JSON breaks on newlines. Python wrapper with `json.dumps` is the safe shape for any body carrying markdown.
- **Village thread voice — stay short.** Three posts in one evening in #459 (bridge-stone, 🦝 react, walk-back, goodnight) was already the top of the budget. Karo's echo of the warm-brick line landed because it was one small image, not extended imagery.
